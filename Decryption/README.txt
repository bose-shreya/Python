README 
Download the zip file and unzip it into the desired folder. Once you have done that, open the terminal, on the command line type cd and go into the folder you have unzipped the files. Once you are in the folder, on the command line type python hw1.py and press enter.

Ex: Let's say I have the unzipped file in a folder called 557 in the Desktop
Open Terminal
cd Desktop
cd 557
python hw1.py
